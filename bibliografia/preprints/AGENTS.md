# AGENTS.md

## Escopo
Armazena preprints e working papers ainda não consolidados como publicação formal.

## O que pertence aqui
- Itens do arXiv, SSRN, bioRxiv e repositórios equivalentes.
- Manuscritos circulando para revisão.

## Requisitos mínimos
- Arquivo original ou ponteiro para URL canônica.
- Arquivo `<id>.md` com repositório de origem, versão, data, autores, resumo e tags.
- Arquivo `<id>.bib` com entrada `@misc`, `@article` provisória ou equivalente.

## Skills preferenciais
- `ai-engineer`
- `trend-researcher`

## Observações
- Se um preprint virar artigo publicado, mover a referência para a categoria final e manter vínculo com a versão anterior.
