# AGENTS.md

## Escopo
Scripts utilitários para triagem, catalogação, validação e geração de relatórios.

## Regras
- Cada script deve ser idempotente sempre que possível.
- Usar caminhos relativos ao repositório; evitar caminhos absolutos hardcoded.
- Expor interface CLI clara e documentada.
- Preferir scripts pequenos, especializados e compostáveis.
- Quando um script alterar arquivos em lote, oferecer modo de simulação ou relatório prévio.

## Skills preferenciais
- `ai-engineer`
- `devops-automator`
- `backend-architect`
