# AGENTS.md

## Escopo
Área de retenção para itens que não podem ser incorporados imediatamente ao corpus.

## Quando usar
- Arquivo corrompido.
- Duplicata não trivial.
- Metadados insuficientes.
- Material fora de escopo.

## Regras
- Não apagar itens desta pasta sem registrar o motivo.
- Para cada item, criar um arquivo `*_ISSUE.md` explicando o problema e o próximo passo sugerido.
- Se o problema for resolvido, mover o item para `../` para reprocessamento ou diretamente para `../../bibliografia/` quando apropriado.
